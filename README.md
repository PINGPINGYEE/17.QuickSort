# 17.QuickSort
### 비교
이동 측면에서는 삽입 정렬이 가장 적게 나왔습니다. 또한 비교 측면에서는 shell sort가 가장 작게 나왔습니다. 하지만 이동과 비교를 통합해서 봤을때는 QuickSort가 가장 효율적이라고 생각이 들었습니다. 비교적 이동횟수도 적은편이고 비교횟수도 적었기 때문입니다.
